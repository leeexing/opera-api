# OPERA_API

> 为运维数据平台提供后台接口

## 项目架构

```conf
app 项目主模块
|
|__conf 配置项模块
|
|__controllers 项目业务控制模块
|
|__models 模型模块
|
|__public 通用模块
|
|__resource 路由模块
|
|__util 工具类模块
|
|__api.py api模块
|
|__db.py 数据库模块
|
|__main.py Flask应用模块
|
|docs 项目文档
|
|manage.py 程序入口
|
|.pylintrc 项目编写规范（Pep8）
|
|requirements.txt 项目所需第三方模块

```

## 技术栈

* Flask
* Flask-Script
* Flask-SQLAlchemy
* Flask-RESTFUL
* Flask-restplus
* Flask-Migrate
* Flask-SocketIO
